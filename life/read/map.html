# 脑图

# 《情感勒索》

《情感勒索》 (opens new window)是苏珊·福沃德经典作品，书中对关系中的情感操控和精神暴力进行了非常细致的介绍并给出了应对方法。

# 《依恋》

《依恋》 (opens new window)是苏珊·福沃德另一部关于情感操控和精神暴力的作品，书中着重介绍了控制性男性对伴侣进行的情绪控制和打压，以及被操控方的女性为何会被操控和陷入，里面牵涉到了原生家庭和依恋理论，在操控手段上与《情感勒索》类似，对女性非常受用。

# 《依恋与亲密关系》

《依恋与亲密关系》 (opens new window)介绍了两性关系的最新治疗方法——情绪取向疗法，该疗法立足于依恋理论，着重关注伴侣间破坏性方式的根源——依恋需求，指出那些破坏性对话的背后其实源于情感联结的断裂，源于依恋需求的不被满足所造成的情感剥夺，它触发了根植于成长关系经历里的负性知觉和情绪记忆，会造成情绪上的不适和内心的不安全感，继而无意识地以防御性攻击的方式进行沟通，从而将两性关系逼入死角，如果了解依恋理论，应该会感觉非常有效和共鸣。

# 《理智胜过情感》

《理智胜过情感》 (opens new window)是我读过的心理治疗类别里最棒的一本书，实操性极强，不管是理念和方法，都感觉非常清晰合理，书里采用的是认知行为疗法，如果你存在强迫，焦虑和抑郁等情绪心理问题，这本书一定能给你带来帮助！六星推荐！

# 《爱的沟通》

《爱的沟通》 (opens new window)是一本从求爱到相爱各个方面都有所阐述的书，主要是针对男性群体，讲述如何与女人恋爱相处，非技巧型求爱书籍，内容正向，对于恋爱经验不足的男性比较有启发作用，我觉得比金星火星系列更直接有效，后者偏向于两性差异的角度。

# 《如何让你爱的人爱上你》

《如何让你爱的人爱上你》 (opens new window)的内容顾名思义，讲的主要是两性间的恋爱吸引，非技巧型求爱书籍，男女都比较适用，能从中学到很多基本的恋爱原则和思维，能够避开很多交往中的异性雷区，帮助你进行自我调整和改善。

# 《吸引力原理》

这个脑图的内容来源于我在知乎上看到的一个两性吸引力专栏，是别人总结的一张脑图

# 《欲望的演化》

《欲望的演化》 (opens new window)源于我看了一些恋爱相关的书籍后，对两性吸引力原理的好奇，因为那些书籍背后我都看到了进行心理学的影子，然后在这本经过科学研究的书籍上进行了一波验证，果不其然。简单而言，所谓吸引力，不过是一种进化出来的择偶策略，它在我们交往的过程中无意识地发挥作用，不过因为人类社会的后期发展速度远远超过了基因演化的速度，导致我们的择偶心理基础发展迟滞，但是它依然在我们的两性交往中发挥了核心作用，尽管我们的理性认知能很聪明地进行择偶选择，但是决定自身喜欢对方与否，对对方是否有感觉的那部分还是取决于过去这种择偶心理和本能，不管人类如何用理性去解释和概括自己的行为，对于没有感觉的人，我们就是很难喜欢，对于想了解两性吸引力根源的朋友非常推荐去看。
