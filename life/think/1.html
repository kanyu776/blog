    # 《对两性关系的理解1》

    下面这些是我在知乎看了一个恋爱专栏和后续阅读进化心理学书籍后得出的一些理解和感想

    # 为什么男性更容易花心/出轨

    从生物学进化论的角度来说，是因为雄性需要保证自身后代的繁衍水平，所以会尽可能多地与雌性进行交配，保证后代的存活率和基因的延续性。

    # 为什么男人会怕带绿帽子

    同样基于生物进化论的角度，是因为雄性害怕无法辨认出自己的孩子，无法准确地延续血脉，保留基因，背后的恐惧和焦虑其实源于生物学本能。

    # 为什么女性比起出轨，更担忧男性不再爱他

    同样基于生物进化论的角度，雄性不再“爱”她，意味着生存能力的重大缺失，需要面临生存风险和危机，这种情感焦虑背后其实是本能焦虑。

    # 生存价值和繁衍价值

    • 生存价值，是我们在这个世界上的生存能力的指标。

    • 繁殖价值，是衡量我们后代基因是否优良的指标。

    • 女性更为突出的是繁衍价值（美貌/身材）。

    • 男性更为突出的是生存价值（金钱/物质）。

    # 两性择偶的本质

    • 其实男性和女性都在无意识地完成同一件事情，那就是生存和繁衍，它们指导着我们的求偶行为。

    • 两性择偶，其实就是对生存价值和繁衍价值的选择。

    • 只是比较而言，男人更看重女人的繁衍价值，女人更看重男人的生存价值。

    # 吸引与爱情

    • 男女之间的吸引，本质上是一种生物性驱力在起作用，它背后的意义是生存和繁衍，是物种和基因的延续。

    • 不过，因为文明给予了这个隐藏的生物性趋力一个很好的目的性和方向感，那便是爱情。

    # 镌刻在我们基因中的生物学影响

    • 男人播种（性猎取），女人选种（备胎），就是这样一种生物学的本能，它们基于远古时代的生存环境，这种几百万年的基因影响远大于近几千年的文明演化。

    • 不管是男人喜欢漂亮丰满的女人（繁殖价值），还是女人喜欢优秀完美的男人（生存价值），其背后的无意识力量都是生物性的，它根植在我们的基因里。

    • 一定程度上，我们克服了生物学本能，但一旦将全部生活种类的意义浓缩到根本问题上，那大概就剩下两件事：生存和繁衍。
